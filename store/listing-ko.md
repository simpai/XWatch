# XMate Store Listing (KO)

## Summary (132 chars)
X 프로필 메모 저장/표시와 예약 시간 단축 버튼을 제공하는 생산성 확장 프로그램

## Description
XMate는 X(구 Twitter)에서 유저별 메모를 기록하고 다시 보는 도구입니다.

주요 기능:
- 프로필 화면에서 유저별 메모 작성/저장
- 타임라인에서 저장한 메모를 이름 옆에 표시
- userId 기반 저장으로 핸들 변경에도 데이터 유지
- 여러 기기 동기화(`chrome.storage.sync`)
- 설정 페이지에서 저장 데이터 조회/검색/삭제
- 게시물 작성 중 한글 IME 입력 잘림을 줄이기 위한 입력 보정
- 예약하기 모달에서 빠른 시간 조정 버튼 제공
  - 5분뒤/10분뒤/30분뒤/1시간뒤
  - +1시간/-1시간/+1일/-1일

데이터 관련:
- 사용자가 직접 입력한 메모 데이터만 저장합니다.
- 데이터는 브라우저 동기화 저장소에 보관되며 별도 자체 서버로 전송하지 않습니다.
